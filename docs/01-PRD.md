# 极简星座/八字流年运势测算 微信小程序 PRD

> 文档版本：V1.0 ｜ 状态：可排期 ｜ 适用范围：MVP 首版 ｜ 日期：2026-04-30
> 适用团队：产品 / 前端（uni-app）/ 后端（SpringBoot + 微信云托管）/ 测试 / 运营
> 本期数据库不连接，全部使用 Mock 假数据，但接口契约必须按真实业务定义。

---

## 1. 结论（先看这段）

做一款"轻量级娱乐型运势测算"微信小程序，主打"3 秒出结果、每日一签、激励视频解锁深度"，通过微信流量主（Banner + 激励视频 + 插屏）变现。本期为 MVP，**只做娱乐测算 + 广告变现闭环**，不做支付、不做会员、不做社区、不做命理咨询。

- **核心价值**：每日打开 1 次，看 30 秒结果 + 1 次激励视频 = 1 次广告收入。
- **过审策略**：全程"娱乐参考"定位，规避封建迷信、生死祸福、医疗健康建议。
- **本期排期建议**：前端 8 人日 + 后端 5 人日 + 测试 3 人日 + 文案库 2 人日。

## 2. 产品定位与用户画像

**一句话定位**：一款主打"每日一签 + 星座八字娱乐解读"的微信小程序，免费、无注册、看广告解锁详细内容，定位为娱乐消遣工具。

- **目标用户**：18-35 岁女性为主（占比 70%+），三线及以下城市用户对运势内容敏感度高；学生、职场新人、宝妈为主力人群。
- **使用场景**：早晨通勤刷一刷今日运势、午休摸鱼、睡前查明日宜忌。
- **过审逻辑**：
  - 全程使用"娱乐参考""仅供消遣"措辞，不涉及具体预测；
  - 所有页面底部固定免责声明；
  - 不出现"算命""改命""化解灾祸""佛""道""神"等敏感词；
  - 不收集真实身份证、不做付费占卜、不导流外部社群。

## 3. 核心页面清单与功能点

### 3.1 首页（Home）

| 模块 | 字段/内容 | 交互 |
|---|---|---|
| 顶部 Banner | 当日日期 + 农历 + 节气 + 一句鸡汤 | 静态展示，文案库轮换 |
| 主 CTA 按钮 | "开始今日测算" | 点击 → 检查是否已录入信息 → 有则进结果页，无则进录入页 |
| 功能卡片区（4 个） | 今日运势 / 八字流年 / 星座匹配 / 幸运签 | 点击进入对应入口（八字流年、星座匹配、幸运签 MVP 锁定，提示"敬请期待"） |
| 每日签到 | "已连续签到 X 天" | 点击签到 → 弹出签到动画 + 1 句吉祥话（本地缓存计数，不入库） |
| 底部导航栏 | 首页 / 历史 / 我的 | Tab 切换 |
| 广告位 | 底部 Banner 广告（原生模板） | 页面加载即展示 |
| 免责声明 | "本内容仅供娱乐参考，请理性看待" | 固定底部小字 |

### 3.2 信息录入页（Profile Input）

| 字段 | 类型 | 是否必填 | 说明 |
|---|---|---|---|
| 昵称 | 文本，≤8 字 | 选填 | 默认"小星友" |
| 性别 | 单选（男/女/不愿透露） | 必填 | "不愿透露"也允许 |
| 公历生日 | 日期选择器（1940-01-01 ~ 当前-1 年） | 必填 | 不允许未来日期 |
| 出生时辰 | 12 时辰下拉（含"不确定"） | 选填 | 默认"不确定" |
| 性格标签 | 多选 chips（最多 3 个） | 选填 | 用于结果文案微调 |
| 协议勾选 | 默认未勾 | 必填 | "我已阅读并同意《用户协议》《隐私政策》《免责声明》" |

### 3.3 运势结果详情页（Fortune Result）

| 模块 | 说明 |
|---|---|
| 顶部摘要卡 | 昵称 + 星座 + 生肖 + 综合评分（0-100，星级 + 数字） |
| 今日宜忌 | 宜：3 条 / 忌：3 条 |
| 四大维度（默认仅展示概要） | 感情 / 财运 / 事业 / 健康，每项一句概要 + "查看详情"触发激励视频 |
| 幸运三件套 | 幸运色 / 幸运数字 / 幸运方位（直接展示） |
| 开运小建议 | 1 段 50 字以内 |
| 解锁深度内容 | 折叠区，激励视频解锁 |
| 分享 / 保存图片 | 两个按钮 |
| 再测一次 | 按钮 |
| 免责声明 | 固定 |

### 3.4 历史记录页（History）

- 倒序排列，最多保留 30 条
- 空状态：插画 + "还没有记录哦" + CTA
- 列表项点击跳转结果详情（只读，不再触发广告）
- 清空记录二次确认
- 列表中每 5 条插入 1 个原生广告

### 3.5 个人中心（Mine）

- 头像区（微信头像/默认 + 昵称）
- 我的资料（编辑入口）
- 连续签到天数
- 设置（订阅消息开关、清除缓存）
- 关于我们
- 协议入口（用户协议/隐私政策/免责声明）
- 意见反馈
- 底部 Banner

## 4. 业务流程

**主流程：首次使用**

1. 打开小程序 → 首页（无需授权即可浏览）。
2. 点击"开始今日测算" → 判断本地是否已存录入信息。
3. 录入页提交 → Loading 1.5s → 调后端 mock → 进入结果页。
4. 默认展示评分、宜忌、幸运三件套、四大维度概要。
5. 点击"查看详情" → 激励视频弹窗 → 完成回调 → 解锁。
6. 离开页面 → 插屏广告（每日 1 次频控）。

**异常分支**：未授权可继续用、未填生日按钮置灰、网络失败兜底文案、激励视频失败不强制解锁。

---

## 5. 边界条件与异常处理

| 场景 | 处理方案 |
|---|---|
| 用户拒绝授权微信信息 | 不阻断主流程，头像默认 |
| 未填生日点击提交 | 按钮置灰 |
| 协议未勾选 | 按钮置灰 + toast |
| 生日选择未来日期 | 前端限制最大值 |
| 用户年龄 < 14 岁 | 弹窗提示 + 屏蔽感情维度 |
| 网络超时 | 5s 后展示重试，3 次失败兜底文案 |
| 重复点击提交 | 按钮置灰 3s + 接口加锁 |
| 激励视频拉取失败 | toast，不解锁 |
| 激励视频中途关闭 | 不解锁 |
| 同日多次测算 | 同日同用户结果一致（hash） |
| 历史满 30 条 | 自动覆盖最早 |
| 分享图片生成失败 | toast + 兜底"长按二维码" |
| 用户清缓存重进 | 视为新用户 |

## 6. 合规红线（违规即下架）

**禁止类**

- 严禁"算命""改命""化煞""消灾""超度""法事"等封建迷信词汇。
- 严禁预测"生死""疾病""手术""车祸""官司"等具体祸福事件。
- 严禁"佛""道""神""菩萨""仙人"等宗教人物。
- 严禁宣称"准确率""灵验""必应"。
- 严禁诱导付费占卜、加微信、加群、跳转外站。
- 严禁针对未成年人推送感情、婚恋类内容。
- 严禁医疗健康维度给具体建议。

**必须类**

- 每个结果页底部固定显示免责声明。
- 启动页/首页/个人中心三处必须有协议入口。
- 文案以"建议""适合""不妨"句式，避免"必将""一定"。
- 感情/财运维度用语温和，不出现"分手""破财""失业""背叛"。

**文案话术正向化样板**

- 感情：今天适合主动表达关心，对方会感受到你的真诚。
- 财运：今天适合复盘消费习惯，节制小额支出。
- 事业：今天适合处理细节事务，沟通时多一点耐心。
- 健康：今天注意补充水分、保持心情愉悦。

## 7. 流量主广告策略

| 位置 | 类型 | 时机 | 频次 |
|---|---|---|---|
| 首页底部 | Banner / 原生模板 | 加载即展示 | 常驻 |
| 结果页摘要下方 | 原生模板 | 进入页面 1s 后 | 每次 1 个 |
| 结果页"查看详情" | 激励视频 | 用户主动点击 | 单页最多 4 次 |
| 结果页离开 | 插屏 | 返回/切 Tab | 每用户每日最多 1 次 |
| 历史列表 | 信息流 | 列表中每 5 条插入 1 个 | 列表常驻 |
| 个人中心底部 | Banner | 加载 | 常驻 |

**激励视频奖励规则**：解锁单个维度详情、明日运势预告。不与签到/积分/虚拟货币挂钩。

## 9. 本期范围 vs 不做项

### 9.1 本期 MVP（必做）
- 5 个核心页面、每日运势 mock 测算、流量主四类广告位、三大协议静态、本地缓存历史/资料/签到、分享/保存图片、订阅消息开关、基础埋点。

### 9.2 不做项（防止 scope creep）
- 不做：真实数据库、支付/会员/付费解锁/积分商城、真人命理咨询/IM/社区/UGC、深度排盘/合婚/紫微、AI 大模型实时生成、多端适配、国际化、复杂用户体系、消息推送服务端、运营后台、A/B 测试。

## 10. 风险点

1. **审核风险（最高）**：建议以"星座娱乐"类目提交，避开"占卜""命理"。
2. **未成年人保护**：感情维度对 <14 岁屏蔽。
3. **广告频控**：激励视频频次过高被判"诱导观看"。
4. **文案风险**：≥90 套文案，敏感词过滤 + 法务/运营双审。
5. **留存风险**：依赖订阅消息授权率，录入页后引导一次。
6. **本期不涉及**订单/支付/库存/会员/优惠券/打印/售后，后期接入需重新评估。
